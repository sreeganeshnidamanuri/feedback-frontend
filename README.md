# Feedback Management System

A full-stack **Feedback Management System** built using **React (Vite) for the frontend** and **Spring Boot for the backend**.
This application allows users to submit feedback and enables administrators to manage, review, and resolve feedback through an admin dashboard.

---

## 🚀 Technologies Used

### Frontend

* React
* Vite
* Axios
* React Router
* React Icons
* React Hot Toast
* CSS

### Backend

* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven

---

## 📌 Features

### User Features

* Submit feedback with **name, email, and message**
* Form validation
* Success/error notifications

### Admin Features

* View all feedback
* Filter feedback by status
* Update feedback status (Pending / Reviewed / Resolved)
* Edit feedback
* Delete feedback
* View feedback statistics
